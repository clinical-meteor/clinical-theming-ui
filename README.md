##clinical:theming-ui

Theming UI component for Clinical Framework.  

===============================
#### Installation  

````bash
meteor add clinical:theming-ui
````

===============================
#### Usage  

````js
Router.go('/theming');
````

````html
{{> themingPage }}
{{> themingCard }}
````
